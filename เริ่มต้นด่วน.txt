📌 วิธีรันโปรแกรมแบบง่าย ๆ
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1️⃣  ดับเบิลคลิกที่ไฟล์:  start.bat
2️⃣  รอโปรแกรมเริ่มต้น (ครั้งแรกอาจใช้เวลาสักครู่)
3️⃣  เปิดเบราว์เซอร์ไปที่:  http://localhost:3000
4️⃣  เริ่มใช้งาน JaiKod.com ได้เลย! 🎉


📂 ไฟล์สำคัญที่ต้องรู้จัก
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✅ start.bat
   → รันโปรแกรมแบบ Development (สำหรับพัฒนา)
   → มีระบบ Hot Reload (รีโหลดอัตโนมัติเมื่อแก้ไขโค้ด)

✅ start-production.bat
   → รันโปรแกรมแบบ Production (สำหรับใช้งานจริง)
   → ประสิทธิภาพสูงกว่า แต่ต้อง Build ใหม่เมื่อแก้ไขโค้ด

✅ install.bat
   → ติดตั้ง dependencies (packages) ที่จำเป็น
   → ใช้เมื่อโหลดโปรเจกต์มาใหม่หรืออัปเดต packages

✅ clean-rebuild.bat
   → ล้าง cache และติดตั้งใหม่ทั้งหมด
   → ใช้เมื่อพบปัญหาแปลก ๆ หรือโปรแกรมทำงานผิดปกติ

✅ open-browser.bat
   → เปิดเบราว์เซอร์ไปที่ http://localhost:3000
   → ใช้เมื่อเซิร์ฟเวอร์รันอยู่แล้ว


⚙️ ข้อกำหนดเบื้องต้น
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📦 Node.js (เวอร์ชัน 18 ขึ้นไป)
   ดาวน์โหลดได้ที่: https://nodejs.org/
   (เลือก LTS version)


🔧 การตั้งค่าครั้งแรก
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. ติดตั้ง Node.js (ถ้ายังไม่มี)
2. แก้ไขไฟล์ .env.local (ใส่ค่า Firebase Config)
3. ดับเบิลคลิก start.bat
4. เปิดเบราว์เซอร์ไปที่ http://localhost:3000


🆘 แก้ไขปัญหาเบื้องต้น
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

❌ ปัญหา: ไม่พบ Node.js
   → ติดตั้ง Node.js จาก https://nodejs.org/

❌ ปัญหา: Port 3000 ถูกใช้งานอยู่
   → ปิดโปรแกรมที่ใช้พอร์ต 3000
   → หรือเปลี่ยนเป็นพอร์ตอื่น

❌ ปัญหา: โปรแกรมทำงานผิดปกติ
   → รัน clean-rebuild.bat

❌ ปัญหา: Firebase Error
   → ตรวจสอบไฟล์ .env.local ว่าใส่ค่าครบถ้วน

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
