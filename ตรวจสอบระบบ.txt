📍 เข้าถึงหน้าตรวจสอบระบบ
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. ไปที่: http://localhost:3000/admin
2. คลิกการ์ด "ตรวจสอบระบบ" (สีน้ำเงิน)

หรือเข้าโดยตรง: http://localhost:3000/admin/system-check


🔍 รายการที่ตรวจสอบ (6 รายการ)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📁 Configuration
  ✓ Environment Variables - ตรวจสอบไฟล์ .env.local

☁️ Firebase
  ✓ Firebase Configuration - ตรวจสอบค่า Config
  ✓ Firestore Database - ทดสอบการเชื่อมต่อ
  ✓ Authentication - ตรวจสอบระบบล็อกอิน
  ✓ Storage - ตรวจสอบที่เก็บไฟล์

📦 Dependencies
  ✓ NPM Packages - ตรวจสอบแพ็กเกจที่ติดตั้ง


🎯 วิธีใช้งาน
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. เปิดหน้า System Check
2. รอระบบตรวจสอบอัตโนมัติ (5-10 วินาที)
3. ดูผลลัพธ์:
   🟢 สีเขียว = ปกติ
   🟡 สีเหลือง = มีปัญหาบางส่วน
   🔴 สีแดง = มีข้อผิดพลาด
4. คลิกแต่ละการ์ดเพื่อดูรายละเอียด
5. คลิก "ตรวจสอบใหม่" เพื่ออัปเดต


🛠️ แก้ไขปัญหาด่วน
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

❌ Environment Variables ผิด
   → แก้ไขไฟล์ .env.local
   → รีสตาร์ทเซิร์ฟเวอร์

❌ Firebase ไม่เชื่อมต่อ
   → ตรวจสอบ Firebase Console
   → ตรวจสอบ Firestore Rules
   → Deploy rules ใหม่

❌ Packages ไม่ครบ
   → รัน install.bat
   → หรือ clean-rebuild.bat
