## 💡 Use Cases ทั้งหมด

### Scenario 1: ผู้ขายลงขายสินค้า

```
1. เข้าหน้า /sell
2. เลือกหมวดหมู่: โทรศัพท์มือถือ
   → AI โหลด Schema
   → แสดงฟอร์มแบบ Dynamic

3. กรอกข้อมูล:
   - ยี่ห้อ: Apple
   - รุ่น: iPhone 14 Pro
   - ความจุ: 256GB
   - สภาพ: มือสอง สภาพดีมาก
   - แบตเตอรี่: 88%

4. กดปุ่ม "AI ประเมินราคา"
   → AI วิเคราะห์
   → ราคาแนะนำ: ฿36,800
   → ช่วงราคา: ฿31,280 - ฿42,320
   → ความมั่นใจ: 85%

5. กดปุ่ม "AI เขียนรายละเอียด"
   → AI สร้างรายละเอียดอัตโนมัติ
   → หัวข้อ + รายละเอียด + แท็ก + SEO

6. กดปุ่ม "ลงขายเลย!"
   → ✅ สำเร็จ!
```

### Scenario 2: ผู้ซื้อค้นหาสินค้า

```
1. เข้าหน้าค้นหา
2. พิมพ์: "หาไอโฟนมือสองราคาไม่เกิน 30000"
   
   → AI แก้ไขคำผิด: "iPhone"
   → AI เข้าใจ:
     - หมวดหมู่: โทรศัพท์
     - แบรนด์: Apple
     - สภาพ: มือสอง
     - ราคา: ≤ 30,000

3. เปิด Location
   → AI ค้นหาสินค้าใกล้เคียง
   → จัดอันดับตาม:
     - ความเกี่ยวข้อง (40%)
     - ระยะทาง (20%)
     - ความชอบส่วนตัว (20%)
     - ความนิยม (10%)
     - ความใหม่ (10%)

4. แสดงผลลัพธ์:
   📱 iPhone 13 Pro 256GB - 2.5km - ฿28,900
   📱 iPhone 12 Pro 128GB - 3.2km - ฿22,500
   📱 iPhone 13 128GB - 1.8km - ฿25,900

5. คลิกดูสินค้า
   → AI บันทึกพฤติกรรม
   → ใช้แนะนำสินค้าในอนาคต
```
